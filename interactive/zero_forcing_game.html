Zero Forcing Game

Colour-change rule: If a blue vertex has exactly one white neighbour, it forces that neighbour to turn blue. All such forcings happen simultaneously in each step. A set of initially-blue vertices is a zero forcing set if repeated application of the rule eventually colours every vertex blue. The zero forcing number Z(G) is the smallest such set.
